# Expense Tracker

Автор: Александра Аркадьевна Николайчук

## Описание
GUI-приложение на Python для учёта личных расходов.

## Возможности
- Добавление расходов
- Просмотр расходов в таблице
- Подсчёт общей суммы расходов
- Фильтрация по категории
- Сохранение данных в JSON
- Проверка корректности ввода

## Использование
1. Запустите программу:
python main.py

2. Введите:
- сумму;
- категорию;
- дату в формате ГГГГ-ММ-ДД.

3. Нажмите «Добавить расход».

4. Для подсчёта расходов нажмите
«Подсчитать сумму».

5. Для поиска введите категорию и нажмите
«Фильтровать».

## Пример
1000 | Еда | 2025-06-20
500 | Транспорт | 2025-06-21
